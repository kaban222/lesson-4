Занятие 4.
Цикл for. Циклы while и do
Перед выполнением заданий ознакомьтесь с примерами выполнения
заданий из лекции «2. Управляющие инструкции: условия и циклы»
(слайды 28-36).
Для всех вариантов дается одинаковое задание, но с разными данными. В
условии задачи также присутствует ответ, которого нужно достичь.
В задачах нельзя использовать структуры для хранения данных (массивы,
списки, вектора и пр.). В задачах нельзя использовать собственные функции.
В задачах нужно провести оптимизацию настолько, насколько это возможно
(вычисление начальных значений, использование подходящего типа данных,
замена долгих вычислений более оптимальными).
Условия задач:
a) Вычислить и распечатать значение суммы ряда. Использовать цикл for.
b) Среди первых 25 элементов последовательности 𝑢𝑖 найти значение
максимального элемента и его номер. Использовать цикл for.
c) Вычислить сумму s (определяется формулой) для x = 2. Провести
оптимизацию формулы. Не использовать готовую функцию вычисления
факториала. Использовать цикл while.
d) Определить наименьший номер элемента последовательности, заданной
рекуррентной формулой, начальными условиями и условием выхода из
цикла. Использовать цикл do .. while.
